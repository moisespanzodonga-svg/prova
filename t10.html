Introdução ao CSS

O que é CSS?

O CSS (Cascading Style Sheets) é uma linguagem utilizada para definir a aparência de páginas web. Ele trabalha em conjunto com o HTML, permitindo separar o conteúdo da apresentação visual.

Destaque: Sem CSS, uma página HTML seria apenas texto e imagens sem qualquer estilo ou layout organizado.

Importância do CSS

O CSS é essencial porque permite:

  • Separar conteúdo e design: facilita a manutenção do site.
  • Melhorar a estética: cores, fontes, tamanhos, espaçamento e posicionamento.
  • Adaptabilidade: torna o site responsivo para diferentes dispositivos (computadores, tablets, smartphones).

Exemplo: Com CSS podemos mudar a cor de um título usando color: green; ou o tamanho da fonte com font-size: 24px;.

Tipos de folhas de estilo

O CSS pode ser aplicado de três maneiras diferentes:

  • Inline: o estilo é adicionado diretamente no elemento HTML usando o atributo style. Exemplo: <h1 style="color: green;">Título</h1>
  • Interno: o estilo é definido dentro de uma tag <style> na própria página HTML.
  • Externo: o estilo é escrito em um arquivo .css separado, que é vinculado à página usando <link rel="stylesheet" href="estilos.css">. Este é o método mais recomendado para sites grandes.

Dica: Usar CSS externo facilita reaproveitar estilos em várias páginas de um mesmo site.

Conclusão

O CSS é uma ferramenta poderosa que transforma páginas HTML simples em sites visualmente atraentes e organizados. Com ele, podemos controlar cores, fontes, layouts, animações e muito mais, garantindo uma experiência agradável para os usuários.

Fundamentos do CSS Moderno

Sintaxe Básica • Selectores • Elementos, Classes e ID

1. Sintaxe Básica do CSS

O CSS (Cascading Style Sheets) é usado para definir o estilo visual de páginas HTML. A sua sintaxe é simples e organizada.

Estrutura básica: 
selector {
propriedade: valor;
}

Exemplo prático:

p {
color: green;
font-size: 16px;
}

Neste exemplo, o seletor p aplica estilo a todos os parágrafos.

2. Selectores Simples

Os selectores indicam quais elementos HTML receberão os estilos definidos no CSS.

  • Selector de elemento: p, h1, div
  • Selector de classe: .nomeDaClasse
  • Selector de ID: #nomeDoId

Exemplo:

h1 {
color: darkgreen;
}

3. Elementos, Classes e ID

No CSS moderno, usamos elementos, classes e IDs para aplicar estilos de forma organizada e reutilizável.

Elementos

Aplicam estilos a todas as tags do mesmo tipo.

p {
color: #374151;
}

Classes

Permitem reutilizar estilos em vários elementos.

.destaque {
background-color: #dcfce7;
padding: 10px;
}

ID

Usado para estilizar um único elemento específico.

#principal {
border: 2px solid green;
}
Dica moderna: Prefira classes em vez de IDs para facilitar manutenção e reutilização do código.

O que são Pseudo-classes?

Pseudo-classes são palavras-chave adicionadas a seletores CSS que especificam um estado especial do elemento selecionado. Elas permitem estilizar elementos com base em sua posição, estado de interação, ou relacionamento com outros elementos, sem precisar adicionar classes extras no HTML.

  • :hover – Ativado quando o usuário passa o mouse sobre o elemento
  • :nth-child() – Seleciona elementos com base em sua posição entre irmãos
  • :first-child – Seleciona o primeiro elemento filho de seu pai
  • :last-child – Seleciona o último elemento filho de seu pai
  • :not() – Seleciona elementos que NÃO correspondem ao seletor
  • :checked – Aplica a elementos de formulário marcados/checados
