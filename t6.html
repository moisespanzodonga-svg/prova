Introdução ao CSS

O que é CSS?

O CSS (Cascading Style Sheets) é uma linguagem utilizada para definir a aparência de páginas web. Ele trabalha em conjunto com o HTML, permitindo separar o conteúdo da apresentação visual.

Destaque: Sem CSS, uma página HTML seria apenas texto e imagens sem qualquer estilo ou layout organizado.

Importância do CSS

O CSS é essencial porque permite:

  • Separar conteúdo e design: facilita a manutenção do site.
  • Melhorar a estética: cores, fontes, tamanhos, espaçamento e posicionamento.
  • Adaptabilidade: torna o site responsivo para diferentes dispositivos (computadores, tablets, smartphones).

Exemplo: Com CSS podemos mudar a cor de um título usando color: green; ou o tamanho da fonte com font-size: 24px;.

Tipos de folhas de estilo

O CSS pode ser aplicado de três maneiras diferentes:

  • Inline: o estilo é adicionado diretamente no elemento HTML usando o atributo style. Exemplo: <h1 style="color: green;">Título</h1>
  • Interno: o estilo é definido dentro de uma tag <style> na própria página HTML.
  • Externo: o estilo é escrito em um arquivo .css separado, que é vinculado à página usando <link rel="stylesheet" href="estilos.css">. Este é o método mais recomendado para sites grandes.

Dica: Usar CSS externo facilita reaproveitar estilos em várias páginas de um mesmo site.

Conclusão

O CSS é uma ferramenta poderosa que transforma páginas HTML simples em sites visualmente atraentes e organizados. Com ele, podemos controlar cores, fontes, layouts, animações e muito mais, garantindo uma experiência agradável para os usuários.
